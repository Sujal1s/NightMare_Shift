
Poneglyph Reader System - Line-by-Line

- Player approaches Poneglyph
- 'Press Y to Read' appears
- Press Y to start reading
- Each press of Y advances to the next line
- Dialogue ends when all lines are shown

Setup:
1. Attach `PoneglyphReader.cs` to your Poneglyph GameObject
2. Set 'promptUI' to your 'Press Y' UI element
3. Set 'dialogueBoxUI' to your dialogue box panel
4. Set 'dialogueTextUI' to the Text component inside the dialogue box
5. Fill 'dialogueLines' with your story text (in the Inspector)
